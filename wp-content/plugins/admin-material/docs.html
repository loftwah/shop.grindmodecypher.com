Admin Theme Documentation

Requirements

  • Wordpress 3.8 or higher

Installation

  1. Download, unpack and copy the "admin-theme" folder into your wordpress site "wp-content/plugins" folder
  2. Login to your wordpress admin dashboard
  3. Go to "Plugins" on the left
  4. Active the "Admin Theme" plugin
  5. Go to "Settings->Admin Theme"

Use it

In Wordpress admin page, click "Appearance"->"Admin Theme"
